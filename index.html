Αυτή είναι μια εφαρμογή όπου μπορείτε να κάνετε μια λίστα με πράγματα που θέλετε να αγοράσετε. Μπορείτε να προσθέσετε στοιχεία στη λίστα πληκτρολογώντας και την τιμή τους και, στη συνέχεια, κάνοντας κλικ σε ένα κουμπί. Ο ιστότοπος θα σας εμφανίσει τη λίστα με τα είδη που καταχωρίσατε και θα υπολογίσει απο την λίστα ποιο αντικείμενο είναι η καλύτερη προσφορά.

Εάν θέλετε να αφαιρέσετε ένα στοιχείο από τη λίστα, μπορείτε να κάνετε κλικ σε ένα κουμπί δίπλα σε αυτό το στοιχείο [Χ]. Εάν θέλετε να αφαιρέσετε όλα τα στοιχεία από τη λίστα, μπορείτε να κάνετε κλικ στο κουμπί [Clear Items].

Ο ιστότοπος θα θυμάται τη λίστα των στοιχείων που εισαγάγατε, επομένως εάν επιστρέψετε στον ιστότοπο αργότερα, θα εξακολουθείτε να βλέπετε τη λίστα σας.
